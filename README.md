# web-portal-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可被视作轻量级的信息入口或功能组件，不针对任何单个域名或特定网站。

## 项目简介

`web-portal-notes` 是一个静态页面集合，旨在以 HTML 格式保存和展示各类笔记、工具页面或快速访问入口。每个页面独立存在，互不依赖，便于单独引用或部署。本项目适合用于存放临时或长期需要保留的 Web 片段。

## 目录说明

仓库根目录下包含以下内容：

- **index.html** — 仓库首页，提供页面列表导航。
- **pages/** — 存放所有独立 HTML 页面的子目录。
- **assets/** — 可选资源目录，用于存放 CSS、JavaScript、图片等公共文件。
- **README.md** — 本文件。

其中 `pages/` 目录下的每个 HTML 文件均代表一个独立的归档页面，文件名应简洁明了，反映页面内容。

## 页面归档说明

新增或修改页面时，请遵循以下约定：

- 每个页面应自包含，避免引用外部不可控资源（除非必要）。
- 页面标题建议使用 `<title>` 标签设置，便于识别。
- 如需引用 `assets/` 下的公共资源，请使用相对路径。
- 页面文件命名建议使用小写字母、数字和连字符（`-`），例如 `example-note.html`。

## 维护说明

- 本仓库不提供动态后端支持，所有页面均为纯静态 HTML。
- 如需更新某个页面，直接修改对应文件并提交即可。
- 若需新增页面，在 `pages/` 目录下创建新 HTML 文件，并视情况更新 `index.html` 的导航列表。
- 建议定期检查页面链接和资源引用是否有效。

## 其他

- 本项目不涉及任何具体域名、网站或商业用途。
- 页面内容由维护者自行决定，仓库仅提供归档和发布功能。
- 欢迎通过 Issue 或 Pull Request 提出改进建议。
